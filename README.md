# Password-Analyzer

\# 🔐 Password Analyzer API



A simple backend project that analyzes password strength and returns a hashed version using bcrypt.



\---



\## 🚀 Features

\- Password strength checker (Weak / Medium / Strong)

\- Detects common weak patterns

\- Generates hashed password using bcrypt

\- REST API built with Express



\---



\## 🛠 Tech Stack

\- Node.js

\- Express.js

\- bcrypt

\- JavaScript



\---
